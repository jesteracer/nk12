h1. Результаты выборов 2011 4 Дек.

Два метода:
1. Для того, что бы забрать данные и записать их в БД (без сохранения html-страниц с сайтов избирательных комиссий) - rake grab:get, в database.yml pool > 18.

2. Для опроса сайтов избирательных комиссий и сохранения html-страниц со всеми данными, необходимыми для заполнения БД, - rake grab:fetch_raw_html.

Для заполнения БД из сохраненных html-страниц, - rake grab:import_to_db_from_raw
Пример директории с первичными данными с сайтов избирательных комиссий (выкачано 22.01.2012) :  http://www.electorus.org/nk12/raw_input.tar.bz2
Размер архива: 63М
Размер после разархивирования: 4.7Гб
